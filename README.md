## 功能与特性

MarketRoute CN 是一款用于 FF14 国服市场批量比价和采购路线规划的 Dalamud 插件。

支持购物清单、HQ 与 NQ 筛选、四大区完整报价对比、服务器路线规划、采购进度记录和跨大区混合分析。

## 安装

在 Dalamud 设置中添加以下第三方仓库：

```text
https://raw.githubusercontent.com/LonelyFSH/MarketRouteCN/main/repo.json
```

保存后，在插件安装器中搜索 `MarketRoute CN` 并安装。

## 使用

输入 `/mrcn` 打开插件，建立清单后即可查询报价、查看路线并开始采购。

## 声明
