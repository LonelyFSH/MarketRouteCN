# 安装

[返回文档首页](README.md) · [返回项目首页](../README.md)

在 Dalamud 设置中添加：

```text
https://raw.githubusercontent.com/LonelyFSH/MarketRouteCN/main/repo.json
```

路径：

```text
/xlsettings
→ Experimental
→ Custom Plugin Repositories
```

保存后打开：

```text
/xlplugins
```

搜索 `MarketRoute CN` 并点击安装。

安装完成后输入：

```text
/mrcn
```
